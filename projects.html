Home Banking Application

My project 'Homebanking Application' is a standout example of my ability to develop robust and functional web applications. I used a variety of cutting-edge technologies, including Java, Spring Boot, PostgreSQL, Postman, Vue.js, HTML, CSS, Gradle, and IntelliJ to create this simulation of a complete banking application.
This application offers users the ability to register, log in, manage bank accounts, make transfers to both their own and third-party accounts, request credit and debit cards, and apply for loans. It is a versatile and secure tool that replicates the experience of using a real banking application.
I am proud of this project as it showcases my expertise in high-quality web application development with a focus on security and usability. Serving as a testament to my commitment to software development excellence, 'Homebanking Application' is a valuable component of my portfolio that demonstrates my technical and creative skills in action.

Amazing Events

My project 'Amazing Events' is an innovative web platform that provides detailed information about events, both upcoming and past. Users can explore upcoming events, learn about their location, time, maximum capacity, and, in the case of past events, the attendance percentage. What makes 'Amazing Events' exceptional is its ability to allow users to filter events effectively.
The platform features two powerful filters that can be used simultaneously, enabling users to fine-tune their searches in a cross-cutting manner. The first filter consists of event categories that can be checked or unchecked based on user preferences. The second filter allows for text-based searches, making it easy to find specific events. The combination of both filters ensures that users find exactly what they are looking for.
To make this functionality possible, JavaScript has been used, adding a layer of interactivity and dynamism to the user experience. 'Amazing Events' not only provides information about events but also enhances the way users interact with it.
This project showcases my ability to create interactive and functional web applications that effectively cater to user needs. 'Amazing Events' is a valuable addition to my portfolio and demonstrates my capability to implement innovative web solutions.

Mighty Ducks Hockey League
